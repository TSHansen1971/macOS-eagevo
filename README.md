# macOS-eagevo

macOS-eagevo er Eagevo Control Plane for macOS.

macOS forblir det tekniske operativsystemet. Eagevo bygges som et lokalt kontrollplan oppå macOS.

Kjerneformulering:

macOS utfører.
Eagevo klassifiserer, vurderer, dokumenterer, auditerer og styrer.

Første fase er foundation og contract-only. Ingen enforcement, ingen AI-runtime, ingen agent-autonomi, ingen ekstern logging og ingen offensive sikkerhetsverktøy skal aktiveres i foundation-fasen.
